## WordPressのテスト環境
### 実行方法
```
docker-compose up -d
```

### コンテナに入る方法
execする。
```
docker exec -it wordpress /bin/bash
```

### wp-cli使用方法
`docker-compose run --rm `の後にコマンドを入力する。
以下は例。
```
docker-compose run --rm wpcli --info
```

## プラグインのスキャフォールド
```
docker exec -it wordpress /bin/bash
wp --allow-root scaffold plugin-tests test-plugin
```
